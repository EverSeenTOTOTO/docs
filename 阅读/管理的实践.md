# 管理的实践

+ 盈利不是企业和企业活动的目的，而是企业经营的限制性因素。利润并不能解释所有的企业活动与决策，而是检验企业效能的指标。

+ 企业认为自己的产品是什么，并不是最重要的事情，对于企业的前途和成功尤其不是那么重要。顾客认为他购买的是什么，他心目中的“价值”何在，却有决定性的影响，将决定这家企业是什么样的企业，它的产品是什么，以及它会不会成功兴旺。

+ 由于企业的目的是创造顾客，任何企业有且只有两项职能：营销和创新。营销和创建产生经济成果，其余的一切都是“成本”。

+ 管理者和员工在本质上没有差别，只有责任上的差别。<Notation type="underline">员工的权利不是来自于管理者授权，而是来自所承担的责任。</Notation>区分高层管理者和基层管理者的惟一维度在于时间。

## 我们的事业是什么？

+ 谁是我们真正的顾客？谁又是我们潜在的顾客？这些顾客在哪里？他们如何购买？如何才能接触到这些顾客？

+ 顾客购买的是什么——由于家庭主妇身为顾客，她们实际上决定了厂商应该生产什么，因此煤气灶制造商真正的事业应该是提供简易的烹调方式，它们的市场是食品烹调市场，竞争对手则是提供各种烹调方式的供应商。

+ 在顾客心目中，价值是什么？顾客采购时究竟在寻找什么？

## 我们的事业将是什么？

+ 市场潜力和市场趋势：假定在市场结构和技术都没有任何基本变化的情况下，5年、10年后，我们预期市场会变得多大？哪些因素会影响市场的发展？

+ 经济发展、流行趋势和品味的变化，或竞争对手的动作，分别会导致市场结构发生什么样的改变？定义“竞争者是谁”的时候，必须以顾客认为他购买的产品和服务是什么为依据，而且也必须包括直接和间接的竞争。

+ 哪些创新将改变顾客需求、创造新需求、淘汰旧需求、创造满足顾客需求的新方式、改变顾客对价值的看法，或带给顾客更高的价值满足感？不仅需要工程技术或化学领域的创新，也需要探讨企业所有的活动领域（不局限于“纯粹的研发”，改善行动能力也是重要任务）。

+ 今天还有哪些顾客需求无法从现有的产品和服务中获得充分满足？能否提出这个问题，并且正确回答问题，通常就是持续成长的公司和只能搭上经济繁荣或产业兴盛的潮流乘势而起的公司真正的差别所在。

## 我们的事业应该是什么？

+ 利润衡量指标的主要用途——防止管理层不断投入资金和心力到衰颓不振的事业，而不是设法强化不断茁壮成长的事业。

+ <Notation type="underline">如果从市场地位、创新和生产力的角度看,跨入新事业领域的决定符合了企业构成的基本条件的话，那么管理层的责任就是要设法获得企业必要的最低利润。</Notation>直言不讳地说，这就是公司付钱聘请管理人员的目的所在。

+ 企业设定目标时，必须以对企业而言最正确而期望的方向为依据，不能为了权宜之计，或顺应经济潮流。也就是说，企业管理不能依赖“直觉”。——依赖直觉的管理者都是企业难以承担的奢侈品。
